# Palletier - 요구사항 정의서

## 1. 개요

AI CLI(Gemini CLI / Claude Code)를 활용하여 사용자가 원하는 느낌의 색상 팔레트를 추천받는 Flutter Desktop 앱이다.

---

## 2. Use Case

| 단계 | 설명 |
|------|------|
| 1 | CLI 실행 파일 위치를 설정한다. 이전에 설정한 경로가 있으면 유지한다. |
| 2 | Model Dropdown에서 사용할 AI CLI를 선택한다. (Gemini CLI / Claude Code) |
| 3 | Auto-detect 버튼으로 CLI를 자동 연결한다. |
| 4 | 자동 연결 실패 시 Manual-detect 버튼으로 CLI 경로를 직접 지정한다. |
| 5 | Login 버튼으로 Credential을 연결한다. |
| 6 | 대화창에 원하는 색상 느낌을 문장으로 입력한다. |
| 7 | AI가 추천한 색상 정보를 대화창에 Markdown 형태로 표시한다. |
| 8 | 왼쪽 스크롤 영역에 추천된 색상 Box들이 가로로 나열된다. |
| 9 | 각 색상 Box 하단의 버튼으로 RGB 코드 복사, 정사각형 이미지 복사, 원형 이미지 복사를 수행한다. |

---

## 3. 기능 요구사항

### 3.1 AI CLI 연결

- **Model 선택**: Dropdown으로 Gemini CLI 또는 Claude Code 중 선택
- **Auto-detect**: 버튼 클릭 시 CLI 실행 파일을 자동 탐지
- **Manual-detect**: 버튼 클릭 시 사용자가 CLI 경로를 직접 지정
- **Login**: Credential 연결 버튼
- **상태 표시**: 연결 상태(Activated / Deactivated) 및 현재 상태 표시
- **경로 유지**: 이전에 설정한 CLI 경로는 앱 재시작 후에도 유지

### 3.2 대화 기능

- 대화창 하단 TextBox에 원하는 색상 느낌을 자연어로 입력
- Commit 버튼(위쪽 화살표)으로 입력 전송
- 추천 색상 개수 선택 Dropdown (3 / 5 / 7)
- AI 응답을 Markdown 형태로 대화창에 표시

### 3.3 색상 팔레트 표시

- 왼쪽 스크롤 영역에 추천된 색상 Box들이 가로로 나열
- 각 Box 구성:
  - **상단 80%**: 해당 색상으로 채워진 영역
  - **하단 20%**: 3등분된 복사 버튼
    - RGB 코드를 클립보드에 복사
    - 해당 색상 배경의 무테두리 정사각형 이미지 복사
    - 해당 색상 배경의 무테두리 원형 이미지 복사

## 4. 화면 레이아웃

### 4.1 전체 구성

```
+---------------------------+----------------+
|                           |  CLI 연결 영역  |
|                           |  (상단 20%)    |
|   색상 팔레트 영역          +----------------+
|   (왼쪽, 가로 스크롤)       |                |
|                           |  대화 영역      |
|                           |  (Markdown)    |
|                           +----------------+
|                           | [TextBox][↑][N]|
+---------------------------+----------------+
```

### 4.2 오른쪽 대화창

- 기본 너비: 전체 너비의 **20%**
- 경계를 드래그하여 너비 조절 가능
- **상단 20%** - CLI 연결 영역:

  | Model Dropdown | Auto-detect | Manual-detect | 상태 표시 |
  |---|---|---|---|
  | Login 버튼 | 현재 상태 | | |

- **하단** - 입력 영역:

  | TextBox (80%) | Commit 버튼 (10%) | 색상 개수 Dropdown (10%) |
  |---|---|---|
  | 자연어 입력 | ↑ 모양  | 3 / 5 / 7 |
